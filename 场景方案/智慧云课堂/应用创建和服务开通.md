# 开通服务
若您之前从未开通过任何网易云信的产品服务，请先在官网首页通过 QQ、在线消息或电话等方式联系[云信商务经理](https://yunxin.163.com/bizQQWPA.html)，一对一沟通您的具体需求。

 1. 登录网易云信控制台。

 2. 在左侧导航栏中单击指定应用名称，进入该应用的详情页面。

 3. 在未开通功能页中找到需要开通的功能项，单击立即开通；部分功能可能需要联系销售进行开通。
